> # 🔑내가 해결한 방법 
> * `git add -u`  &nbsp;&nbsp;&nbsp;  (= update track file 최신화 하겠다)
> * `git commit -m "아무거나 치세요"`
> * `git push`
>
><br>
>
> * `git rm --cached -r .`    (로컬저장소에 자료 남겨두고 원격저장소에 있는 자료만 삭제하겠다)\
> = 내 노트북의 폴더 자료 살리고, 깃허브 인터넷상 자료만 삭제할께 ㅎ
> * `git commit -m "아무거나 치세요"`
> * `git push`
> 
> <br>
>
