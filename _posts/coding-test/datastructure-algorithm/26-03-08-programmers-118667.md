# [[2022 KAKAO TECH INTERNSHIP] 두 큐 합 같게 만들기](https://school.programmers.co.kr/learn/courses/30/lessons/118667)


## 🙋‍♂️ 들어가며
이번 문제는 총 2가지 풀이다.

`queue`

그리고

`투포인터`
<br>
<br>
<br><br><br>


먼저 queue로 풀때는 다음 조건들을 생각해야겠다.

1. q1, q2를 설정하고 합이 일치할때까지 구한다
2. (sum1, sum2) % 2 == 1 이면 -1 을 반환하고 종료
3. 단 짝수일때도 무한루프가 걸릴 수 있는걸 유념하고, 2번 조건만 보지는 말자. 이어서 최악의 케이스를 생각해보자
    
    `3-1.` 아래 2개의 큐를 가정하자, $q_1$ 그리고 $q_2$

    이때 sum1을 $q_1$의 합, sum2를 $q_2$의 합으로 가정
    ```java
    q1 = {a1, a2, a3, a4, a5}
    q2 = {b1, b2, b3, b4, b5}
    ```

    `3-2.` 만약 (sum1 > sum2) 라서 $q_1$의 원소를 전부 옮겨도 답이 안나오면
    ```java
    q1 = {}
    q2 = {b1, b2, b3, b4, b5, a1, a2, a3, a4, a5}
    ```
    이때 연산 -> n회

    `3-3.` 다시 $q_1$로 옮긴다면?
    ```java
    q1 = {b1, b2, b3, b4, b5, a1, a2, a3, a4, a5}
    q2 = {}
    ```

    이때 연산 -> 2n

    `3-3.` 이때 q1에서 4회만 옮긴다면?
    ```java
    q1 = {b5, a1, a2, a3, a4, a5}
    q2 = {b1, b2, b3, b4}
    ```

    이때 연산 근사치 -> n

    만약 $b_5$를 한번 더 옮기면 각 배열의 맨 처음상태로 원상복구가 된다.
    
    즉 모든 경우를 합하여 4n이 최악의 실행연산이므로, 4n을 넘어가는 연산은 없겠다
4. sum1 과 sum2가 같지 않을때까지만 계속 계산을 수행하자, 3번조건을 limit으로 설정하여 말이다


## ✅ 코드 : queue

```java
import java.util.Queue;
import java.util.LinkedList;

class Solution {
    public int solution(int[] queue1, int[] queue2) {
        int cnt = 0;
        
        Queue<Integer> q1 = new LinkedList<>();
        Queue<Integer> q2 = new LinkedList<>();
        
        long sum1 = 0;
        long sum2 = 0;
        
        int N = queue1.length;
        for (int i = 0; i < N; i++) {
            q1.add(queue1[i]);
            sum1 += queue1[i];
            
            q2.add(queue2[i]);
            sum2 += queue2[i];
        }
        
        // 1. 조기종료
        if ( (sum1 + sum2) % 2 == 1) return -1; 
        
        
        // 2. 수행
        int limit = (queue1.length + queue2.length) * 2;
        while (sum1 != sum2) {
            
            // limit
            if (cnt > limit) return -1;
            
            if (sum1 > sum2) {
                int cur_val = q1.poll();
                sum1 -= cur_val;
                q2.add(cur_val);
                sum2 += cur_val;
            }
            else if (sum1 < sum2) {
                int cur_val = q2.poll();
                sum2 -= cur_val;
                q1.add(cur_val);
                sum1 += cur_val;
            }
            
            // 횟수 추가
            cnt++;
        }
        
        
        return cnt;
    }
}
```
