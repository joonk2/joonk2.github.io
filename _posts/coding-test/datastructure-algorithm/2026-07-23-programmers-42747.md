# [[프로그래머스/LV2] H-Index](https://school.programmers.co.kr/learn/courses/30/lessons/42747)


## 🙋‍♂️ 들어가며
정렬을 먼저 해보자 그리고 어떻게 해야 H-Index를 구하는지도 보고
```java
citations = {0, 1, 3, 5, 6};
idx = {0, 1, 2, 3, 4};
```

아 H-Index = citations길이 - i 겠구나.



<br>

## ✅ 정답 코드
```java
import java.util.Arrays;

class Solution {
    public int solution(int[] citations) {
        int answer = 0;
        int N = citations.length;
        
        Arrays.sort(citations);
        for (int i = 0; i < N; i++) {
            int h = N - i;
            if (citations[i] >= h) {
                answer = h;
                break;
            }
            
        }
        return answer;
    }
}
```
