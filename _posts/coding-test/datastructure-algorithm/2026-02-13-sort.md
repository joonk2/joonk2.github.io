## 🤔 시작
```java
int[] arr = {3,1,5,1,6,2,7,3,3};
```
이런 배열이 있다고 가정하자

그 이후 cnt배열을 만들어 배열의 크기는 max_num + 1까지 만들고

arr의 숫자만큼 칸을 채워주자 그러면 다음과 같을 것이다
```java
int[] cnt = {0, 2, 1, 3, 0, 1, 1, 1}
```
그다음 제일 중요한 누적합을 해줘야한다

그러면 다음과 같을 것이다

```java
int[] cnt = {0, 2, 3, 6, 6, 7, 8, 9};
```

## ✅ 정답
```java
int[] arr = {3,1,5,1,6,7,2,3,3};
int N = arr.length;

// 1. 최댓값 구하기
int max = 0;
for (int i = 0; i < N; i++) {
    if (arr[i] > max) max = i;
}

// 2. 카운팅 배열
int[] cnt = new int[max + 1];

// 2-1. 카운팅 배열 초기 추가
for (int i = 0; i < N; i++) {
    cnt[ arr[i] ]++;
}


// 2-2. 카운팅 배열 누적합 적용
for (int i = 1; i < cnt.length; i++) {
    cnt[i] += cnt[i-1];
}


// 3. 오름차순 정렬
int[] res = new int[N];
for (int i = N-1; i >= 0; i--) {
    int cur_val = arr[i];
    res[ cnt[ arr[i] ] - 1 ] = cur_val;
    cnt[ arr[i] ]--;
}

```

res = {1, 1, 2, 3, 3, 3, 5, 6, 7};
